# Where to Edit Content & Assets

## Site Info
`src/config.ts` - Edit site title, author, description, website URL

## Homepage Content
`src/content/about/about.md` - Edit the about section shown on homepage

## Logo & Icons
- `src/config.ts` - Set `site.logo` to show logo on homepage (e.g., `'/logo.png'`)
- `public/favicon.svg` - Browser favicon
- `public/apple-touch-icon.png` - Apple touch icon
- `public/og/og-logo.png` - Logo for social sharing images
- `public/og/og-bg.png` - Background for social sharing images

## Blog Posts
`src/content/posts/` - Add/edit blog posts here
